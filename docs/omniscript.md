# OmniScript

Criar um OmniScript que guie o usuário no processo de registro de tickets de suporte.

## Criação do OmniScript
- Nome do OmniScript: `RegistroDeTicketOmniScript`
- Type: `Guided Flow`
- SubType: `TicketRegistration`
- Language: `Português`
